Instalando programas automaticamente!


Winget

O Winget é um gerenciador de pacotes da Microsoft disponível para Windows 10 e Windows 11 que permite baixar e instalar milhares de programas através de linha de comando. Usando ela, você pode criar um script para executar o download e a instalação automática de todos os programas que você usa em seu dia a dia.

Winstall

O Winget é uma ótima opção para automatizar o download e a instalação dos programas no Windows, mas alguns usuários podem ter um pouco de dificuldade para criar os scripts. Winstall é um site que cria o script de instalação para você, basta selecionar os programas que deseja instalar em seu PC e ele irá criar o script.

WinGetUI

WinGetUI é uma das melhores opções para quem não tem familiaridade com linha de comando. Ele oferece uma interface gráfica para o Winget que torna a instalação dos programas muito fácil. Ao ser executado ele carrega a lista de programas e tudo o que você precisa fazer é clicar no programa que deseja instalar no PC

Ninite

O Ninite é um serviço gratuito capaz de automatizar o processo de download e instalação de programas no Windows. Basta acessar o site e selecionar os programas que deseja instalar. Ao fazer isso, ele irá criar um instalador, que ao ser executado baixa e instala automaticamente todos os programas que você selecionou.

Silent Install Helpe

Silent Install Helper é uma ferramenta gratuita capaz de instalar automaticamente múltiplos programas em dispositivos Windows. Ele permite que você crie uma lista de programas que deseja instalar no computador, e, em seguida, instala automaticamente todos os programas da lista para você com a configuração padrão.

Guinget

Guinget é mais um programa gratuito que oferece uma interface amigável para você instalar programas no Windows 10 e 11 usando o Winget. Ao ser iniciado, ele carrega a lista de programas, e você precisa apenas escolher os programas que deseja instalar para ele baixar e instalar automaticamente os programas no PC.
