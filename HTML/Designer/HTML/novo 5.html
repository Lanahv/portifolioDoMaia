Padrão Facade

O padrão Facade é um padrão de projeto de software que fornece uma interface simplificada para um sistema complexo. Isso pode ser útil para ocultar a complexidade do sistema dos usuários finais, tornando-o mais fácil de usar.

Benefícios

  • Simplificação: O padrão Facade simplifica o uso de um sistema complexo, ocultando a complexidade do sistema dos usuários finais.:
  • Encapsulamento: O padrão Facade encapsula a complexidade do sistema, tornando-o mais fácil de entender e manter.:
  • Extensibilidade: O padrão Facade pode ser usado para estender um sistema sem modificar a sua estrutura.:

Exemplos de uso deste padrão:

  • Desenvolvimento de jogos
    • Ocultar a complexidade de um motor de jogo
      • Unreal Engine
      • Unity
    • Fornecer uma interface simples para um sistema de física
      • PhysX
      • Havok
    • Fornecer uma interface simples para um sistema de animação
      • MotionBuilder
      • Maya
  • Desenvolvimento de hardware
    • Ocultar a complexidade de um sistema operacional
      • Windows
      • Linux
    • Fornecer uma interface simples para um driver de dispositivo
      • NVIDIA GPU Driver
      • AMD Radeon Driver
    • Fornecer uma interface simples para um sistema de controle de processos
      • Linux Process Control API
      • Windows Process Control API
