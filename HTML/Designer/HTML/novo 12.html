Padrão Command

O padrão Command é um padrão de projeto de software que permite que você encapsule uma solicitação em um objeto. Isso pode ser útil para desacoplar o solicitante do objeto que executa a solicitação.

Benefícios

  • Decoupling: O padrão Command desacopla o solicitante do objeto que executa a solicitação. Isso torna o sistema mais flexível e fácil de manter.
  • Encapsulamento: O padrão Command encapsula a solicitação em um objeto. Isso torna a solicitação mais fácil de entender e usar.
  • Testabilidade: O padrão Command torna a solicitação mais testável. Isso ocorre porque a solicitação pode ser testada independentemente do objeto que a executa.
