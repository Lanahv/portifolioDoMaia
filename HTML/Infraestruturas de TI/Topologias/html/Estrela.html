Estrela

O que é uma topologia do tipo Estrela?

A topologia do tipo Estrela (Star) - Refere-se à rede em que todos os dispositivos são conectados a um único ponto central, chamado de hub ou switch. O hub ou switch é responsável por retransmitir os dados entre os dispositivos.

Para que serve a topologia do tipo Estrela?

  • Conectar computadores
  • Conectar dispositivos
  • Conectar periféricos

Onde usar a topologia do tipo Estrela?

  • Comumente usada em redes domésticas
  • Pequenas empresas
  • Redes de campus

Como implementar a topologia do tipo Estrela?

Para implementar uma topologia em estrela, é necessário um hub ou switch e cabos de rede. Os cabos de rede são conectados entre os dispositivos e o hub ou switch.

Ferramentas a topologia do tipo Estrela?

Existem várias ferramentas disponíveis para ajudar na implementação de uma topologia em estrela. Algumas das ferramentas mais comuns incluem:

  • Ferramentas de crimpagem de cabos, usadas para crimpar os conectores dos cabos de rede.
  • Ferramentas de teste de cabos, usadas para testar a conectividade dos cabos de rede.
  • Ferramentas de configuração de hubs e switches, usadas para configurar os hubs e switches.
